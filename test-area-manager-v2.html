🚨 Troubleshooting

If loading fails or shows errors:

  1. Check console for error messages from RoleBasedDataLoading
  2. Verify area manager has assigned stores in store_management_hierarchy
  3. Check if Clerk user ID matches users table
  4. Ensure supabaseAdmin connection is working

If Detail View doesn't work:

  1. Check if preloadedData is properly structured
  2. Verify WeeklyPlanInterface receives combined data
  3. Look for amendment processing logs
  4. Check if store hierarchy data is complete
